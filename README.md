# Innovation
🎉创新创业教育平台项目

项目基于Vue CLI重新架构，需要卸载旧版的vue-cli，本项目统一使用node.js 10.15.3(LTS)

## 安装Vue-CLI
```
npm install -g @vue/cli
```

## 安装本项目所需依赖

```
npm install axois echarts element-ui iview --seve
```

## 项目启动
```
npm install
```

### 编译与热加载 (development)
```
npm run serve
```

### 编译与打包 (production)
```
npm run build
```
