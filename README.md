# Innovation
🎉创新创业教育平台项目

项目采用前后端分离架构，基于Vue CLI，需要卸载旧版的vue-cli，本项目统一使用node.js 10.15.3(LTS)

#### 本项目使用的部分依赖 element-ui iview echarts axois 

### 安装Vue-CLI
```
npm install -g @vue/cli
```

### 项目启动
```
npm install
```

### 编译与热加载 (development)
```
npm run serve
```

### 编译与打包 (production)
```
npm run build
```
